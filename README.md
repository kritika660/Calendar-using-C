📅 Calendar Application (C Language)
📌 Project Description

This project is a console-based calendar application developed in C.
It allows users to view calendars, navigate between dates using keyboard keys, and display calendars for individual months or the entire year.
The project demonstrates the use of structures, loops, conditionals, functions, and date calculations.

🎯 Features

Display calendar for any given month and year

Arrow key navigation for moving between dates/months

Full-year calendar view

Correct handling of leap years

Colored text output for better readability

Works for years from 1945 onwards

User-friendly console interface

🛠️ Technologies Used

Language: C

Compiler: GCC / MinGW

Platform: Windows (Console-based)

⚙️ How to Run the Project

Clone the repository:

git clone https://github.com/your-username/calendar-c-project.git

Navigate to the project folder:

cd calendar-c-project

Compile the program:

gcc calendar.c -o calendar

Run the executable:

calendar
